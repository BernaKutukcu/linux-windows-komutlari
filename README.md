# Linux ve Windows Komut Rehberi

Linux ve Windows işletim sistemlerinde sık kullanılan komutları 
kategorilere ayırarak derleyen bir referans rehberi.

## İçerik

- [Linux Komutları](./linux_komutlari.md) — dosya işlemleri, ağ, 
  sistem yönetimi, paket yönetimi ve daha fazlası (60+ komut)
- [Windows Komutları](./windows_komutlari.md) — dosya işlemleri, 
  ağ, sistem ve disk yönetimi (60+ komut)

## Kullanım Amacı

Terminal veya komut satırı (CMD) kullanırken hızlı başvuru 
kaynağı olarak tasarlanmıştır.
