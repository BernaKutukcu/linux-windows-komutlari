# Windowsta En Sık Kullanılan 50+ Komut
Bu dosyada Windows işletim sisteminde sıkça kullanılan komutlar ve açıklamaları listelenmiştir.

## 📁 Dosya ve Klasör İşlemleri

- `dir` : Bulunduğun klasördeki dosya ve klasörleri listeler.

- `cd` : Çalışma dizinini değiştirir.

- `cd ..` : Bir üst klasöre çıkar.

- `mkdir` : Yeni klasör oluşturur.

- `rmdir`: Boş klasörü siler.

- `del` : Dosya siler.

- `copy` : Dosya kopyalar.

- `xcopy` : Klasör ve alt klasörleri kopyalar.

- `robocopy` : Gelişmiş kopyalama/senkronizasyon yapar.

- `move` : Dosya veya klasörü taşır.

- `ren` : Dosya veya klasör adını değiştir.

- `attrib` : Dosya özniteliklerini görüntüler veya değiştirir (salt okunur, gizli vb.).

## 🌐 Ağ Komutları

- `ipconfig` : IP adresi, ağ geçidi ve alt ağ maskesi gibi ağ bilgilerini gösterir.  

- `ping` : Belirli bir adrese ulaşılıp ulaşılamadığını test eder.  

- `tracert` : Hedefe giden yol boyunca geçen sunucuları gösterir.  

- `netstat` : Aktif bağlantıları ve açık portları gösterir.  

- `nslookup` : Alan adlarının IP adreslerini gösterir (DNS sorgusu yapar).  

- `arp -a` : ARP önbelleğini (IP ↔ MAC adres eşleştirmelerini) listeler.  

- `netsh` : Gelişmiş ağ yapılandırmalarını yönetmek için kullanılır. 

- `net` : Kullanıcı, paylaşım ve servis yönetimi yapar (net user, net share vb.).

- `whoami` : Mevcut kullanıcı adını ve etki alanını gösterir.

- `gpupdate` : Grup ilkelerini yeniler.

- `gpresult` : Uygulanan grup ilkelerini raporlar.

- `route` : Yönlendirme tablosunu görüntüler veya düzenler.

- `path` : Çalıştırılabilir dosya arama yolunu (PATH) gösterir veya değiştirir.

## 💻 Görüntüleme ve Arama

- `type` : Metin dosyasının içeriğini gösterir.

- `more` : Uzun çıktıları sayfa sayfa gösterir.

- `find` : Dosya içinde metin arar.

- `findstr` : Gelişmiş metin arama (regex destekli).

- `fc` : İki dosyayı karşılaştırır.

- `cls` : Komut satırını temizler.

- `echo` : Ekrana metin yazar veya değişken atar.

## ⚙️ Sistem ve Kullanıcı

- `set` : Ortam değişkenlerini görüntüler/ayarlar.

- `setx` : Ortam değişkenlerini kalıcı olarak ayarlar.

- `chcp` : Komut satırı kod sayfasını değiştirir.

-  `assoc` : Dosya uzantılarını ilişkilendirir.

- `ftype` : Dosya türüne karşılık gelen programı ayarlar.

- `start` : Program veya pencere başlatır.

- `tasklist` : Çalışan işlemleri listeler.

- `taskkill` : Belirtilen işlemi sonlandırır.

- `wmic` : Sistem bilgilerini WMI üzerinden verir.

- `systeminfo` : Donanım ve OS bilgilerini gösterir.

- `hostname` : Bilgisayar adını gösterir.

- `ver` : Windows sürümünü gösterir.

- `driverquery` : Yüklü sürücüleri listeler.

## 🛠 Onarım ve Disk İşlemleri

- `sfc` : Sistem dosyalarını tarar/onarır. 

- `dism` : Windows bileşenlerini onarır. 

- `chkdsk` : Disk hatalarını tarar/onarır.

- `diskpart` : Disk bölümlerini yönetir.  

- `format` : Sürücüyü biçimlendirir.

- `label` : Sürücü etiketini değiştirir.

- `vol` : Sürücü etiketini ve seri numarasını gösterir.  

- `defrag` : Disk birleştirme yapar.

- `cipher` : NTFS şifreleme ve güvenli silme işlemleri yapar. 

- `icacls` : Dosya/klasör izinlerini yönetir.  

## 🔌 Güç ve Yardım

- `shutdown` : Bilgisayarı kapatır/yeniden başlatır.

- `schtasks` : Zamanlanmış görev oluşturur.  

- `powershell` : PowerShell’i başlatır.  

- `help` : Komutlar hakkında yardım gösterir. 

- `exit` : Komut isteminden çıkar.
